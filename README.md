# Titanic Dataset EDA (Python)

This project was completed during the IIT Madras School Connect Program.

This repository contains my analysis and code screenshots for exploring the Titanic "Machine Learning from Disaster" dataset. The project focuses on conducting Exploratory Data Analysis (EDA) using Python to understand patterns related to passenger survival.

## Project Focus
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
